# SlothLabs Smart Contracts

## Overview

This directory contains the smart contracts powering SlothLabs' crowdfunding platform.

**Status:** Pre-audit development

## Security Notice

⚠️ **These contracts have NOT been audited.** Do not deploy to mainnet without:
- Professional security audit
- Comprehensive unit tests
- Extended testnet deployment (30+ days)
- Bug bounty program
